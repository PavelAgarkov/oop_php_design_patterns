Шаблоны находятся в директории src/patterns и разделены по категориям.
В директории каждого шаблона имеется скрипт demonstration.php.
Скрипт демонстрации index.php находится в папке src.
Для запуска демонстрации конкретного шаблона нужно раскомментировать подключение соответвующего файла demontration.php из директории шаблона в src/index.php.

Пакет помогает разобраться с основными шаблонами проектирования приложений на php.
